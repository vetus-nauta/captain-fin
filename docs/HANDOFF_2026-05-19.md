## What Exists

- iOS-style light web app shell.
- Notes-first report editor.
- Opens the first non-submitted report; if none exist, opens a new blank report.
- Report list acts as the web app home page.
- Desktop/tablet layout keeps the original pattern: left list, right working column.
- Admin login uses BRKOVIC credentials via `https://brkovic.ltd/api/auth/login`.
- Guests and crawlers get no indexed content.
- Reports and exports are grouped by year.
- Excel export is created in `storage/exports/YYYY/`.
- Export attempts async rclone copy to Google Drive folder `1x9m41AUYPocx7H0UezF_lZnFvzWO54zQ`.
- `CAPTAIN_FIN_MARKER.json` marks the latest version for localhost mirrors.

## Known Operational Notes

- Server needs PHP with `curl` for live admin auth.
- Server needs PHP `ZipArchive` for true `.xlsx` export.
- Server-side Google Drive duplication needs `rclone` available to the PHP process. If hosting does not allow `exec`, duplication should be handled by cron or local sync.
- The app intentionally stores no admin password.
